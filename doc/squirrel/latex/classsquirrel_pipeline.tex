\doxysubsection{Detailed Description}
The pipeline class. 

pipelines contain 3 sections\+: pipeline info, dataspec, steps (the script)

\mbox{[}Ni\+DB\mbox{]} fields are used when sharing pipelines between Ni\+DB instances. These are optional in the squirrel format spec, but the fields can be useful when sharing pipelines in other contexts
