Detailed Description

The pipeline class.

pipelines contain 3 sections: pipeline info, dataspec, steps (the script)

[NiDB] fields are used when sharing pipelines between NiDB instances. These are optional in the squirrel format spec, but the fields can be useful when sharing pipelines in other contexts
